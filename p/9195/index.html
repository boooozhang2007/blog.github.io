为你的博客更换鸿蒙字体

发表于2023-06-11|更新于2023-06-11|技术
|字数总计:9805|阅读时长:59分钟|阅读量:|评论数:

前言

最近觉得我博客的Apple 字体看腻了，换个鸿蒙字体尝尝鲜，并且打算一直用下去，所以就有了这个教程文章，我Hexo博客用的主题是Butterfly，所以教程自然是围绕着Butterfly去的，不过其他框架或主题的网站应该也能用，毕竟实现原理就是用CSS从CDN引入字体文件而已，还是蛮简单的，琢磨下就会了，没什么技术含量。不废话了，干正事！

教程

0x0001

首先往你的CSS文件内添加以下字体引入代码…

这里用的是bilibili的 CDN 引入的，毕竟阿 B 的 CDN 雀氏又快又好用~，就怕以后有防盗链之类的(～￣ ▽ ￣)～

当然，也可以把src:路径换成自己的，可写绝对路径，也可写相对路径，

此时字体应该就引入成功了，如果是非 Butterfly 主题就不用下一步了。

下载 HarmonyOS Sans

0x0002

打开Butterfly 主题配置文件，自行比较以下配置差异，并进行修改。

小建议：可直接搜索font-family关键字进行快速跳转至需修改的配置项。

font:和blog_title_font:里的font-family:配置项内添加HarmonyOS_Regular

0x0003

然后直接 Hexo 三连后查看效果就完成了！

如果不生效可能是 CSS 引入字体的位置不对，请添加在 CSS 文件的末端，让其覆盖，

如果还是不行就得考虑重装 Hexo 框架与 Butterfly 主题了。

结尾

如果有其他问题可评论留言，有空看到的话一般都会解答！溜了溜了~
